## Notes
This project was collaboratively developed as part of AI-based game development work.# AI-Games
A collection of fun and interactive AI-powered games and creative projects built with Python &amp; OpenCV - includes air canvas, rock paper scissors, hand gesture games, AI drawing guesser, and more.

---

## 📁 Repository Structure

```
AI-Games-and-Fun/
└── project/
   ├── main.py
   ├── requirements.txt
   └── README.md
```

> Each project has its own folder with its own README and requirements file.

---

## 🛠️ Tech Stack

- Python 3.x
- OpenCV
- MediaPipe
- TensorFlow / scikit-learn
- Pygame (for game UI projects)

---

## 🚀 Getting Started

1. Navigate to any project folder
   cd air-canvas

2. Install dependencies
   pip install -r requirements.txt

3. Run the project
   python main.py

---

## 📌 Requirements

- Python 3.8 or above
- Webcam (for gesture and camera-based games)
